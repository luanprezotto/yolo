Sobre a Empresa

Em 2023, quando tive a ideia de criar uma nova bebida refrescante para se refrescar no verão estávamos cansados de beber, os mesmo energéticos de sempre, então, tivemos a ideia de criar nosso próprio energético

Curiosidade:

O nome da nossa empresa, "YOLO", se originou através de uma música, achamos a palavra legal e fomos ver o que significava Em Inglês, YOLO é uma sigla para "You Only Live Once", que traduzindo para o português significa "Você Só Vive Uma Vez" Então, usamos a palavra como forma de celebrar a vida!
